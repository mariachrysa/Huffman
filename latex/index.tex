\hypertarget{index_Introduction}{}\doxysection{Introduction}\label{index_Introduction}
The Huffman coding implementation is a command-\/line program for data compression and decompression. It offers four main functionalities through different command-\/line options\+: probability calculation for characters in a sample text file (-\/p), Huffman code generation based on probabilities (-\/s), file encoding using Huffman codes (-\/e), and file decoding (-\/d). The program employs modular design, with separate files handling probability calculation, tree node structures, encoding and decoding processes. It features a convenient Makefile for compilation, simplifying the build process. Overall, it\textquotesingle{}s a tool for lossless data compression and decompression using the Huffman coding algorithm.\hypertarget{index_Command-Line}{}\doxysection{Argument Guide}\label{index_Command-Line}
-\/p $<$sample.\+txt$>$ $<$probfile.\+txt$>$\+: calculates the probabilities of each ASCII character in a \textquotesingle{}sample.\+txt\textquotesingle{} file and prints them to \textquotesingle{}probfile.\+txt\textquotesingle{}

-\/s $<$probfile.\+txt$>$\+: generates Huffman codes for ASCII characters based on their probabilities from \textquotesingle{}probfile.\+txt\textquotesingle{} which are printed both onto the screen and onto an additional \textquotesingle{}codes.\+txt\textquotesingle{} file
