\doxysubsection{Detailed Description}
Implements data structures and functions for Huffman coding using a Min\+Heap-\/based binary tree. 

This file provides the structures and operations to build and manipulate a Huffman tree for efficient data compression. The implementation includes structures such as \mbox{\hyperlink{structMinHNode}{Min\+HNode}} and \mbox{\hyperlink{structMinHeap}{Min\+Heap}}, representing nodes and the \mbox{\hyperlink{structMinHeap}{Min\+Heap}} data structure. Various operations, including node creation, heap creation, heap maintenance, and tree building, are defined to support the Huffman coding algorithm. Additionally, functions for reading probabilities from a file, generating Huffman codes, and encoding and decoding files are provided.
