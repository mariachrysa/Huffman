Detailed Description

Huffman coding implementation for data compression and decompression.

This program provides a command-line interface for performing various Huffman coding operations, including probability calculation, Huffman code generation, file encoding, and file decoding. The main functionality is encapsulated in the functions defined in this file, and the program follows a modularized structure to promote clarity and reusability.
